## Common Objections Reference

| Objection | When to Use | Sample Language |
|-----------|-------------|-----------------|
| Relevance | Request seeks irrelevant information | "Objection. This request seeks information that is not relevant to any claim or defense and is not reasonably calculated to lead to the discovery of admissible evidence." |
| Overbroad | Request too expansive | "Objection. This request is overly broad and unduly burdensome in that it seeks [explanation]." |
| Vague | Terms undefined or unclear | "Objection. This request is vague and ambiguous as to the meaning of [term]." |
| Privilege | Attorney-client or work product | "Objection. This request seeks information protected by the attorney-client privilege and/or work product doctrine." |
| Burden | Disproportionate to needs | "Objection. The burden of responding to this request outweighs its likely benefit considering the needs of the case and the amount in controversy." |
| Time Period | Unreasonable time frame | "Objection. The time period requested is overly broad and not reasonably related to the issues in this case." |

## Document Production Checklist

**Before Production:**
- [ ] All responsive documents gathered
- [ ] Documents reviewed for privilege
- [ ] Privilege log prepared (if withholding)
- [ ] Documents Bates-stamped
- [ ] Documents organized by request

**Production Format:**
- [ ] PDF copies (preferred)
- [ ] Organized by request number
- [ ] Index/log of documents produced
- [ ] Bates numbering consistent

**Quality Check:**
- [ ] Confidential information redacted (if appropriate)
- [ ] No privileged documents included
- [ ] All responsive documents included
- [ ] Production complete

---

## Completion Checklist

**Response Preparation:**
- [ ] All requests answered
- [ ] Objections stated with specificity
- [ ] Responses verified under oath (interrogatories)
- [ ] Documents gathered and produced
- [ ] Privilege log prepared (if needed)

**Service:**
- [ ] Responses served by deadline
- [ ] Documents produced
- [ ] Certificate of service filed
- [ ] Copy retained for file

**Calendar:**
- [ ] Supplementation reminders set
- [ ] Note any discovery disputes
- [ ] Follow-up on objected items

## Risk Flags

- [ ] Response deadline missed
- [ ] Incomplete responses
- [ ] Missing documents
- [ ] Privilege waiver risk
- [ ] Objections not specific enough
- [ ] RFA deemed admitted (if no response)
