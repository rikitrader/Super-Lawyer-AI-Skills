## Instructions

The Civil Case Information Sheet is a required filing in Texas courts. This guide assists in completing the official form accurately for divorce cases.

---

## Section-by-Section Completion Guide

### Section 1: Cause Number

| Field | Entry |
|-------|-------|
| Cause Number | {{CAUSE_NO}} |
| Notes | Leave blank if filing new case (clerk assigns) |

### Section 2: Styled

| Field | Entry |

### Section 3: County

| Field | Entry |
|-------|-------|

### Section 4: Case Category

**For Divorce Cases, Select:**

| Category | Check |
|----------|-------|
| Civil | ☐ |
| Family Law | ☒ |
| Tax | ☐ |
| Probate/Guardianship/Mental Health | ☐ |

### Section 5: Case Type

**Family Law Case Types:**

| Code | Case Type | Use When |
|------|-----------|----------|
| DIVORCE | Divorce | All divorce cases |
| DIVC | Divorce with Children | Divorce with minor children |
| DIVNC | Divorce without Children | Divorce without minor children |

**Select:**
- ☒ DIVC - Divorce with Children (if minor children)
- ☒ DIVNC - Divorce without Children (if no minor children)

### Section 8: Pending Discovery Plan

| Level | Description | Check |
|-------|-------------|-------|
| Level 1 | Expedited | ☐ |
| Level 2 | Standard (default) | ☒ |
| Level 3 | Complex (by court order) | ☐ |

*Most divorce cases are Level 2*

### Section 9: Monetary Relief Sought

**For Divorce Cases:**

| Amount Range | Check If |
|--------------|----------|
| $0-$100,000 | ☒ (most cases) |
| $100,001-$200,000 | ☐ (moderate assets) |
| $200,001-$1,000,000 | ☐ (substantial assets) |
| Over $1,000,000 | ☐ (high net worth) |

*This is an estimate and does not limit recovery*

### Section 10: Non-Monetary Relief Sought

**For Divorce Cases, typically check:**

- ☒ Divorce/Annulment
- ☒ Custody/Conservatorship (if children)
- ☒ Visitation/Access (if children)
- ☒ Child Support (if children)
- ☐ Spousal Maintenance (if requested)
- ☒ Property Division
- ☐ Injunctive Relief (if TRO/temporary orders)
- ☐ Other: _____________

## Common Errors to Avoid

| Error | Correct Practice |
|-------|------------------|
| Wrong case type code | Use DIVC or DIVNC |
| Missing attorney information | Complete all fields |
| Incorrect discovery level | Level 2 is standard |
| Unsigned form | Must be signed by attorney or pro se party |
| Missing county | Include filing county |

## Electronic Filing Notes

**For E-Filing:**
- Complete form electronically before upload
- PDF format required
- Must accompany original petition
- Separate document from petition

## Completion Checklist

- [ ] Cause number (blank for new filing or assigned number)
- [ ] Parties correctly styled
- [ ] County correct
- [ ] Family Law category selected
- [ ] Correct case type (DIVC/DIVNC)
- [ ] Petitioner information complete
- [ ] Respondent information complete
- [ ] Discovery level indicated (Level 2)
- [ ] Monetary relief range selected
- [ ] Non-monetary relief items checked
- [ ] Form signed by attorney/party
- [ ] Date included

## Risk Flags

- [ ] Pro se party (different requirements)
- [ ] High net worth (may affect court assignment)
- [ ] Complex discovery anticipated
- [ ] Multiple types of relief sought
