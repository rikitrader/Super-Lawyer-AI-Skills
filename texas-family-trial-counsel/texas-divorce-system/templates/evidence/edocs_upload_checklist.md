## eDOCS SYSTEM OVERVIEW

**Website:** https://www.cclerk.hctx.net/

**System Requirements:**
- Modern web browser
- PDF files (preferred format)
- File size limits apply
- Account registration required

**Filing Hours:**
- E-filing accepted 24/7
- Documents filed after 11:59 PM considered filed next business day

## IV. SENSITIVE DATA REQUIREMENTS

### A. Required Redactions (TRCP 21c)

**Social Security Numbers:**
- [ ] Redact all but last 4 digits (XXX-XX-1234)
- [ ] Check all pages for SSNs

**Financial Account Numbers:**
- [ ] Redact all but last 4 digits
- [ ] Includes bank accounts, credit cards, retirement accounts

**Driver's License Numbers:**
- [ ] Redact all but last 4 digits

**Dates of Birth (Minors):**
- [ ] Verify only year shown, or fully redacted

**Home Addresses (if applicable):**
- [ ] Redact if protective order or safety concern
