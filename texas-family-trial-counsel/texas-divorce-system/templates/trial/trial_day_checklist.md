### Pre-Departure

- [ ] Professional attire

- [ ] Review key points
- [ ] Leave with extra time

**Timing:** Arrive {{ARRIVAL_TIME}} (at least 30-60 min before docket)

### Courtroom Setup

## DURING TRIAL

### Docket Call

When case called:
- [ ] Stand
- [ ] Announce appearance: "Ready for [Petitioner/Respondent], Your Honor. [Name] appearing for [Client], who is present."
- [ ] Note any preliminary matters
- [ ] Request bench conference if needed

### Opening Statement

- [ ] Deliver opening
- [ ] Maintain eye contact with judge
- [ ] Stay within time limit
- [ ] Don't argue—state what evidence will show

### Case Presentation

**For Each Witness:**

*Before calling:*
- [ ] Witness ready and present
- [ ] Exhibits for witness organized
- [ ] Examination outline ready

*During direct:*
- [ ] Cover all topics
- [ ] Introduce exhibits properly
- [ ] Listen to answers
- [ ] Note need for redirect

*During cross:*
- [ ] Listen carefully
- [ ] Object when appropriate
- [ ] Note impeachment opportunities
- [ ] Prepare redirect

**Exhibit Tracking:**

| Exhibit | Offered | Objection | Ruling | Notes |
|---------|---------|-----------|--------|-------|
| P-1 | ☐ | | | |
| P-2 | ☐ | | | |

| P-4 | ☐ | | | |
| P-5 | ☐ | | | |

### Cross-Examination of Opposing Witnesses

- [ ] Listen to direct carefully
- [ ] Note inconsistencies
- [ ] Cross only on important points
- [ ] Get admissions, then stop
- [ ] Don't argue with witness

### Closing Argument

- [ ] Review notes from trial
- [ ] Reference specific evidence
- [ ] Cite specific testimony
- [ ] Apply facts to law
- [ ] Request specific relief
- [ ] Thank the Court

### Court's Ruling

- [ ] Take detailed notes
- [ ] Note all rulings on each issue
- [ ] Ask for clarification if unclear
- [ ] Request findings on record

---

## POST-TRIAL

### Immediately After

- [ ] Obtain signed order (if ruling from bench)
- [ ] Note any deadlines
- [ ] Debrief with client
- [ ] Collect materials

### If Favorable Ruling

- [ ] Prepare final decree per ruling
- [ ] Submit to court and opposing counsel
- [ ] Schedule entry
- [ ] Celebrate appropriately with client

### If Unfavorable Ruling

- [ ] Document ruling fully
- [ ] Advise client calmly
- [ ] Discuss options (motion for new trial, appeal)
- [ ] Note deadlines:
  - Motion for new trial: 30 days
  - Appeal: 30 days (or 90 if MNT filed)

### Administrative

- [ ] File any post-trial documents
- [ ] Send client summary letter
- [ ] Update case file
- [ ] Calendar post-trial deadlines
- [ ] Bill time

## COURTROOM REMINDERS

**Do:**
- Stand when addressing the Court
- Address judge as "Your Honor"
- Be courteous to all
- Stay organized
- Listen carefully
- Take notes
- Control your client

**Don't:**
- Interrupt
- Argue with judge
- Be disrespectful to opposing counsel
- Let emotions drive decisions
- Make speaking objections
- Chew gum
- Let phone ring
