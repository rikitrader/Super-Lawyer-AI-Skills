## Quick Start

### Step 1: Clone the Repository

```bash
git clone https://github.com/rikitrader/Super-Lawyer-AI-Skills.git
cd Super-Lawyer-AI-Skills
```

### Step 2: Run the Unlock Script

```bash
./scripts/unlock.sh
```

### Step 3: Enter Your Password

When prompted, enter the password provided by the repository owner.

### Step 4: Skills Auto-Install

The skills will be automatically installed to `~/.claude/skills/` and will activate when you start Claude Code.

## Troubleshooting

**"Wrong password" error:**
- Verify you have the correct password from the repository owner
- Passwords are case-sensitive

**"Permission denied" error:**
```bash
chmod +x ./scripts/unlock.sh
```

**"OpenSSL not found" error:**
```bash
# macOS
brew install openssl

# Ubuntu/Debian
sudo apt-get install openssl
```
